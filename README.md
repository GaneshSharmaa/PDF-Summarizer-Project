# Document Summarizer

Hey Guys! I've built this document summarizer, this summarizes various types of documents like PDF, TXT, URLs.

I've built this project for learning and getting my hands dirty, I believe in learning by doing.

There are various concepts that this project has helped me to have a solid grasp on.

- initializing the LLM
- creating a prompt template — this includes creating system prompts, etc.
- formatting the in-prompt values (placeholders)
- loading the documents, and processing each of them

This project was built on free-tier open source model accessed using GroqAI's API.

FYI, I've used _Meta-Llama-4-Scout 17 billion_ parameter model.

Right now, this project is under development, and only works in the terminal, but soon this will be full-stack application. Deployed.

Soon, more features will be added to this project, as I learn more and more concepts.

Stay tuned! 🙌
